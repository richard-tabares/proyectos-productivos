# Proyectos Web

Plataforma visual que reúne y exhibe los proyectos web desarrollados por los estudiantes del curso de Desarrollo de Software y Tecnología. Sirve como portafolio colectivo donde cada equipo puede presentar su trabajo, compartir el enlace del proyecto en vivo y el repositorio de código.

Los visitantes pueden explorar los proyectos, filtrarlos por categoría y año, y acceder directamente a las demostraciones y al código fuente de cada uno.
